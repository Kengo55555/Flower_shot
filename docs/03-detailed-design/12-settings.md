# 12. 設定画面

依存関係: 03（認証フロー）

---

## 概要

テーマカスタマイズ、位置情報のデフォルト設定、ストレージ容量の確認、管理者画面へのアクセス、ログアウト機能を提供する。

**v2.0変更点**: 漢字表記（大人向け）に変更。テーマカスタマイズ（背景色・ボタン色・花アイコン）を追加。管理者画面へのリンクを設定画面内に配置。PWAインストール状態の表示は削除。

---

## 実装タスク

### 12.1 設定画面（src/pages/SettingsPage.tsx）

- [x] 画面レイアウト（セクション区切りのカード形式、**漢字表記**）
  - セクション1: アカウント
  - セクション2: 背景色
  - セクション3: ボタンの色
  - セクション4: タイトルの花アイコン
  - セクション5: 位置情報
  - セクション6: ストレージ
  - セクション7: 管理者（管理者メールのみ表示）
  - セクション8: アプリ情報
  - ログアウトボタン

### 12.2 アカウント情報セクション

- [x] Googleアカウントアイコン + 表示名 + メールアドレス

### 12.3 テーマカスタマイズセクション（v2.0追加）

- [x] 背景色パレット（7色: レモン / クリーム / 桜 / 空 / 緑 / ラベンダー / 白）
- [x] ボタン色パレット（6色: ピンク / 赤 / オレンジ / 緑 / 青 / 紫）
- [x] 花アイコン選択（8種: 🌻 / 🌸 / 🌷 / 🌹 / 🌺 / 💐 / 🌼 / 🏵）
- [x] タップで即時反映（useTheme フック経由）
- [x] 選択中の項目はborder強調 + スケールアップ

### 12.4 位置情報設定セクション

- [x] 「場所を記録する（既定）」トグル
- [x] トグル変更時に Firestore を更新
- [x] 説明文: 「ONにすると撮影場所を地図に記録できます」

### 12.5 ストレージ情報セクション

- [x] 表示内容（漢字表記）:
  - 📸 保存済み: {N}枚（{usageMB}MB）
  - 📦 残り: 約{N}枚（{remainMB}MB）
  - 🌸 本日の判定: {N}回 / 100回
- [x] 保存に関する制約事項（ボックス内に表示）
  - 写真はこの端末内にのみ保存されます
  - 最大 約500枚まで保存可能（約1GB）
  - 別の端末では写真を閲覧できません
  - アプリを削除すると写真も消去されます

### 12.6 管理者セクション（v2.0変更）

- [x] 「管理者画面 →」ボタンで `/admin` に遷移

### 12.7 アプリ情報セクション
