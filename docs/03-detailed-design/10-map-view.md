# 10. マップ表示

依存関係: 07（保存機能）

---

## 概要

撮影した花の位置を地図上にピンで表示する。ピンタップで花の情報と写真をポップアップ表示する。

**v2.0変更点**: フィルタを月別から年別に変更。ポップアップに写真サムネイルを追加。カスタムピンアイコン（📍絵文字）を使用。

---

## 実装タスク

### 10.1 Leaflet導入

- [x] `npm install leaflet react-leaflet @types/leaflet` でインストール
- [x] Leaflet CSS のインポート
- [x] Leaflet のデフォルトアイコン画像のパス問題を修正（Vite環境）

### 10.2 マップ画面（src/pages/MapPage.tsx）

- [x] 画面レイアウト
  - ヘッダー: 「おはな ちず」タイトル
  - 年フィルタ: 「ぜんぶ」+ 年別ボタン
  - 地図: 画面の残り全体を使用（`h-[100dvh] pb-16`）

- [x] 全ピンが収まるboundsに自動フィット（FitBoundsコンポーネント）

- [x] attribution を表示（Leaflet デフォルト）
