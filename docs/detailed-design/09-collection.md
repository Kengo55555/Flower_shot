# 09. 図鑑一覧

依存関係: 07（保存機能）

---

## 概要

これまでに撮影・保存した花をグリッド形式で一覧表示する。年フィルタとPDFアルバム出力機能付き。

**v2.0変更点**: 年フィルタ追加、PDFアルバム出力ボタン追加。

---

## 実装タスク

### 9.1 図鑑一覧画面（src/pages/CollectionPage.tsx）

- [x] 画面レイアウト
  - ヘッダー: 「ずかん」タイトル + 累計種類数
  - 年フィルタ: 「ぜんぶ」+ 年別ボタン（記録のある年のみ表示）
  - PDFアルバム出力ボタン
  - グリッド: 2列のカード表示（FlowerCard）
  - 空の場合: 励ましメッセージ + 撮影ボタンリンク
- [x] データ取得: `useRecords()` フックで取得
- [x] ソート: `capturedAt` 降順

### 9.2 図鑑カード（src/components/collection/FlowerCard.tsx）

- [x] 画像サムネイル・花名・日付を表示
- [x] 画像がない場合はプレースホルダー表示
- [x] タップで `/detail/{recordId}` へ遷移

### 9.3 年フィルタ

- [x] 記録の年を抽出してボタン生成（新しい年が先）
- [x] 「ぜんぶ」ボタンで全件表示
- [x] 選択中のボタンはピンク背景でハイライト
- [x] フィルタは種類数カウントにも連動

### 9.4 重複排除（種類数カウント）

- [x] 累計種類数は `flowerNameOriginal` の一意数でカウント
- [x] 表示: 「ぜんぶで {uniqueCount}しゅるい みつけたよ！」

### 9.5 PDFアルバム出力

- [x] 「アルバムを つくる（PDF）」ボタン（記録がある場合のみ表示）
- [x] 年フィルタと連動（選択中の年 or 全件）
- [x] 生成中は進捗表示（「{current} / {total} まいめ...」）
- [x] 完了後にブラウザのダウンロードが自動開始
- [x] 詳細は 17-pdf-album.md を参照
